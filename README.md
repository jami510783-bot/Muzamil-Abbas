# MathDetective AI 🔍📐

**MathDetective AI** is an intelligent mathematics tutor powered by Google Gemini AI. It analyzes student solution attempts step-by-step, pinpoints exact errors, explains underlying mathematical misconceptions, provides guided hints without giving away the answer directly, and generates custom practice challenges tailored to student learning levels.

---

## ✨ Key Features

- 🔍 **Step-by-Step Diagnostic**: Upload or enter math problem steps to locate the exact line where a mistake occurred.
- 💡 **Misconception Analysis & Smart Hints**: Get conceptual explanations and progressive hints to guide students toward the right solution.
- 📐 **Interactive Math Rendering**: Rich KaTeX and LaTeX rendering for clean mathematical formulas and equations.
- 🎯 **Tailored Practice Challenges**: Generate custom practice problems targeting specific areas of difficulty.
- ⚡ **Full-Stack Performance**: Express backend integrated with Google Gemini 2.5 AI SDK and a responsive React frontend.

---

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS, Motion, Lucide Icons
- **Math & Markdown Rendering**: KaTeX, `react-markdown`, `remark-math`, `rehype-katex`
- **Backend**: Node.js, Express, `tsx`
- **AI Integration**: `@google/genai` (Google Gemini API)
- **Build Tool**: Vite & esbuild
